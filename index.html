MODELO DE COLAS M/M/S

Sea:

λ= Tasa promedio de llegadas en la unidad de tiempo.

\(\frac{1}{\lambda} \)= Tiempo entre llegadas de los clientes.

μ= Tasa promedio de servicio.

\(\frac{1}{\mu} \)= Tiempo de servicio.

ρ= Tiempo esperado que el servidor este ocupado.

Ws = Tiempo estimado de espera en el sistema. \(W_s=W_q+\frac{1}{\mu}\)

Wq = Tiempo estimado de espera en la fila. \(W_q=\frac{L_q}{\lambda}\)

Ls = Numero estimado de clientes en el sistema. \(L_s=L_q+\rho\)

Lq = Número de clientes en ela fila. \(L_q=\frac{\rho^{S+1}}{\left ( S-1 \right )!\left ( S-p \right )^{2}} P_0 \)

\(P_0\) =Posibilidad de que el sistema este desocupado. \(P_0=\frac{1}{\sum_{n=0}^{n=S-1}\frac{\rho^{n}}{n!}+\frac{\rho^{s}}{S!}\left (\frac{1}{1-\rho/s} \right ) }\)
